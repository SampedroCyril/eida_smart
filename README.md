# eida SMART : the smart app to manage your money.

eida SMART is a web application, the goal was to develop an application which will be used by the eida's clients for managing their money in an easy way.

## Technologies

- MAVEN
- SPRING BOOT
- SPRING SECURITY
- THYMELEAF
- JAVA 
- HTML
- CSS
- JAVASCRIPT
- JUSTGAGE
- VENDOR
- BOOTSTRAP
- UBUNTU MATE 20.04
